# Generate Stellar ABI and Bytecode

## Build the Project

### Project Build

If you want to simply build your Stellar project during development process:

```bash
stellar contract build --optimize
```

Build artifacts are generated in `target/` directory. The `.wasm` file will be generated there, which contains the WASM bytecode of your smart contract

### Contract ABI and Bytecode

Your Stellar bytecode is stored in the `target/` directory. When the build process finishes, you can find it at `/target/wasm32v1-none/release/<contract-name>.wasm`.

:::info
For Stellar, you do not need to generate ABIs like other blockchain systems. Stellar ABIs are included inside Stellar bytecodes already.
:::
