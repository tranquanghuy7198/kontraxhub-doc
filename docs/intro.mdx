# Introduction

<PageLink id="home" /> is a developer tool for blockchain engineers to deploy, explore, and interact with smart contracts — all in one place.

## What is <PageLink id="home" />?

<PageLink id="home" /> provides a clean, unified interface for working with smart contracts across multiple blockchain networks. Instead of juggling CLI tools, block explorers, and custom scripts, you get a single workspace where you can deploy contracts, call their functions, and inspect their state in real time.

Whether you're building on **NEAR**, **Stellar**, **Solana**, or other supported networks, <PageLink id="home" /> adapts to your environment so you can focus on writing contracts, not tooling.

## Key Features

- **Multi-network support** — switch between blockchain networks without changing your workflow
- **Contract Explorer** — inspect deployed contracts, browse their ABI, and call functions interactively
- **Workspace** — organize your contracts into workspaces for cleaner project management
- **Deploy & Interact** — deploy new contracts or connect to existing ones with minimal setup

## Getting Started

### What you'll need

- A supported blockchain account (NEAR, Stellar, Solana, etc.)
- Your compiled contract bytecode or an existing contract address

### Add your first contract

Head over to the <PageLink id="contracts"/> and add a contract by either:

- **Deploying a new contract** — upload your compiled bytecode and deploy directly from the UI
- **Connecting to an existing contract** — provide a contract address to start interacting immediately

### Organize with Workspaces

Use <PageLink id="workspaces"/> to group related contracts together. This is especially useful when working on projects that span multiple contracts across the same or different networks.

More networks are being added continuously.
